# 変更履歴

このアプリにはこれまでバージョン番号がありませんでした。2026-07-12 のセッションで
`v1.2.0` から番号管理を開始し、それ以前の状態を遡って `v1.0.0` / `v1.1.0` として
記録しています。バージョンはヘッダーのアプリ名の横（`#brandVersion`）に表示され、
`js/app.js` 冒頭の `APP_VERSION` 定数が唯一の情報源です。

## v1.2.0（2026-07-12）

- ヘッダーのアプリ名横にバージョン番号を表示する機能を追加（このエントリ自体の変更）

## v1.1.0（2026-07-12・コミット `b124a41`）

認知心理学・情報アーキテクチャ・色彩調和論の観点でのレビューに基づく改良。

- **データ→グラフ到達の導線**: ファイル読込直後の生データ確認画面（生データマトリクス）に
  「この内容でグラフを見る →」ボタンを追加。読込時に自動生成済みのグラフ設定へ、
  明示的なワンクリックで到達できるようにした（それまではステッパーの色変化のみが手がかりだった）
- **グラフ種類の視覚選択**: 単一系列のとき、グラフの種類（棒・折れ線・面・散布）を
  形のアイコンで選ぶピッカーを設定の最上部に追加。複数系列では従来どおり
  系列行ごとの `select` で種類を混在指定する方式に自動で切り替わる
- **列チップの見切れ修正**: `.column-chip` に `min-width: 0` を追加。長い列名が
  「…」省略されずサイドバー境界でそのまま切れていたバグを修正
- **コントラスト改善**: ライトモードの `--muted` を `#898781` → `#6b6a64` に変更し、
  ヒント文・プレースホルダーのコントラストを 3.4:1 → 5.2:1 に改善（WCAG AA達成）
- **状態表現の区別**: ステッパーのホバー色を選択中（`accent-wash`）と異なる中立色に変更し、
  ホバー中と選択中を色だけで見分けられるようにした

## v1.0.0（ベースライン・コミット `2dfd096`）

上記セッション開始前の状態。主な機能:

- CSV・TSV・XLSX・XLS の読み込み、ヘッダー抽出・型推論、生データマトリクスでの手動調整
- フィルタ・グループ化・集計・並び替えをSQL風プレビュー付きのクエリとして構築
- 棒・折れ線・面・散布図の混在描画（最大8系列）、ヒストグラム（階級自動決定・正規分布カーブ）
- 基準線（平均・中央値・±3σ・任意値）、積み上げ表示、グラフ上ドラッグでの範囲ズーム
- グラフ／テーブル／統計の3ビュー、PNG保存・CSV出力
- 処理内容一式を保存・書き出し・読み込みできるプリセット機能
- 検証済みカテゴリカル8色パレット（色覚多様性対応）、ライト/ダークテーマ
