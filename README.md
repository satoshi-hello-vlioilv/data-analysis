# グラフ作成スタジオ — 汎用データ可視化ツール

CSV・Excel ファイルから多彩なグラフを作成・分析できる、ブラウザ完結型の汎用可視化ツールです。
`index.html` を開くだけで動作します（ライブラリ同梱・オフライン対応、データは外部送信されません）。

## 主な機能

### データ入力
- **サンプルデータ内蔵** — 月次売上 / 部品寸法測定 / 店舗別売上 / 気温と販売数 の4種をワンクリックで読込
- **CSV・TSV・XLSX・XLS 対応** — ドラッグ＆ドロップまたはファイル選択
- **ヘッダー抽出処理** — 前置きタイトル行のスキップ、ヘッダー行の自動判定、列名の補完・重複解消、
  列ごとの型推論（数値 / 日付 / テキスト。カンマ付き数値・和暦区切りの日付も解釈）

### データ処理のクエリ化
- フィルタ（列の型に応じた演算子）・グループ化・集計（合計 / 平均 / 中央値 / 最大 / 最小 / 件数 / 標準偏差）・並び替え・行数上限
- すべての処理内容を **SQL風クエリとしてリアルタイム表示**、コピーして共有・再現可能

### グラフ
- **複数系列の同時描画**（最大8系列）と **種類の混在**（棒・折れ線・面・散布図を1つのグラフに）
- **ヒストグラム** — 階級自動決定（スタージェス / スコット / フリードマン・ダイアコニス / カスタム）、
  複数列の重ね合わせ、正規分布カーブ、管理限界線（CL / UCL / LCL）表示（範囲外の基準線まで階級を自動延長）
- **基準線** — 平均・中央値・±3σをワンクリック追加、任意値も設定可
- 積み上げ表示、X位置に吸着する十字カーソル＋全系列ツールチップ

### ビューとエクスポート
- **グラフ / テーブル / 統計** の3ビュー。テーブルはグラフと同一内容（アクセシブルな等価表示）
- 統計ビュー: 件数・平均・中央値・標準偏差・最小・最大・歪度・尖度・分布の形の言語化
- PNG保存・CSV出力（BOM付きでExcel対応）

## デザイン原則

- **色彩工学** — 検証済みカテゴリカル8色（色覚多様性下での隣接色差 ΔE ≥ 12、ライト/ダーク別選定）。
  色は系列（エンティティ）に固定され、系列を削除しても残りの色は変わりません
- **認知心理学 / 情報アーキテクチャ** — 「1. データ → 2. 処理 → 3. グラフ」の段階的開示、
  即時フィードバック（全操作がグラフ・クエリ・テーブルへ同時反映）、データ形状からの賢い初期設定
- **一貫したマーク** — 細い棒（最大24px・データ端のみ角丸）、2px線、表面色リング付きマーカー、
  ヘアラインの控えめなグリッド、単一Y軸（二重軸は作らない）

## 構成

```
index.html      画面構造
styles.css      デザイントークン（ライト/ダーク）と UI スタイル
js/palette.js   検証済みカラーパレット
js/data.js      ファイル解析・ヘッダー抽出・型推論・サンプルデータ
js/query.js     クエリエンジン（フィルタ/集計/SQL風プレビュー）
js/charts.js    グラフモデル構築と Chart.js 描画
js/app.js       状態管理と UI の結線
vendor/         Chart.js v4 / PapaParse / SheetJS（同梱）
```
